# Proyecto 11: Simulación de Decoherencia

Simulación realista de decoherencia cuántica en sistemas abiertos mediante operadores de Lindblad.

## Características

- Simulación T1 (relajación de energía)
- Simulación T2 (dephasing)
- Eco de Hahn (refocalización del dephasing)
- Ecuación maestra de Lindblad
- Visualización de dinámicas

## Parámetros

- T1: tiempo de relajación
- T2: tiempo de dephasing
- Tasas de decaimiento Γ

## Experimentos

1. Decoherencia desde $|+\rangle$
2. Decoherencia desde $|0\rangle$
3. Eco de Hahn (refocalización)
4. Comparación de cohencia

## Uso

```bash
python run_decoherencia.py
```

## Resultados

- Visualización de dinámicas de decoherencia
- Efecto del eco de Hahn
- Tasas de decaimiento
