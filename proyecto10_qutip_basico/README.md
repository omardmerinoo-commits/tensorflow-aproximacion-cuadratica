# Proyecto 10: Simulador Cuántico Básico

Simulación de sistemas cuánticos simples usando QuTiP.

## Características

- 6 estados básicos: $|0\rangle$, $|1\rangle$, $|+\rangle$, $|-\rangle$, $|+i\rangle$, $|-i\rangle$
- Operadores de Pauli: I, X, Y, Z, H
- Esfera de Bloch y coordenadas
- Evolución temporal bajo Hamiltoniano
- Fidelidad y entropía de von Neumann

## Operaciones

- Aplicación de operadores
- Evolución temporal de estados
- Cálculo de valores esperados
- Análisis de entrelazamiento

## Uso

```bash
python run_qutip_basico.py
```

## Visualización

- Esfera de Bloch en 3D
- Evolución de observables
- Gráficas de dinámica temporal
