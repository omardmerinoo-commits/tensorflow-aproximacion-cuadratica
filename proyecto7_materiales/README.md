# Proyecto 7: Predicción de Propiedades de Materiales

Red neuronal para predecir propiedades físicas de materiales (dureza, conductividad, módulo elástico) basada en su composición química.

## Características

- Generador de composiciones químicas sintéticas (5 elementos)
- Predicción de 3 propiedades: dureza, conductividad, módulo elástico
- Red neuronal con arquitectura personalizable
- Validación cruzada

## Uso

```bash
python run_materiales.py
```

## Dataset

- 500 muestras de composiciones sintéticas
- Elementos: Fe, Ni, Cu, C, Cr
- Propiedades simuladas mediante ecuaciones empíricas
