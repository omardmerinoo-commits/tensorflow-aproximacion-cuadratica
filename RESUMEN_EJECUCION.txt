1. PROBLEMA SOLUCIONADO
====================================================================

Error Original:
  ValueError: Could not deserialize 'keras.metrics.mse' because 
  it is not a KerasSaveable subclass

Causa:
  - Incompatibilidad del formato HDF5 (.h5) con Keras 3
  - Cambios en la serialización de métricas en Keras 3

Solución:
  - Cambio de formato HDF5 → .keras (nativo de Keras 3)
  - Actualización de método guardar_modelo() en modelo_cuadratico.py
  - Actualización de método cargar_modelo() en modelo_cuadratico.py
  - Actualización de run_training_fixed.py con nuevas rutas

4. RESULTADOS DE ENTRENAMIENTO
====================================================================

Modelo: Red Neuronal Feedforward
Arquitectura:
  - Entrada: 1 neurona (valor x)
  - Capa Oculta 1: 64 neuronas, ReLU
  - Capa Oculta 2: 64 neuronas, ReLU
  - Salida: 1 neurona, Lineal (predicción y)

Configuración:
  - Optimizador: Adam (learning_rate=0.001)
  - Función de Pérdida: MSE (Mean Squared Error)
  - Métrica: MAE (Mean Absolute Error)
  - Epochs: 100 (con Early Stopping)
  - Batch Size: 32

Datos de Entrenamiento:
  - Muestras: 1000
  - Rango de x: [-1, 1]
  - Ruido: Gaussiano (σ=0.02)
  - División: 80% entrenamiento, 20% prueba

5. COMO USAR EL MODELO
====================================================================

Opción 1: Usar el modelo entrenado directamente
─────────────────────────────────────────────
python
from modelo_cuadratico import ModeloCuadratico
import numpy as np

# Cargar modelo
modelo = ModeloCuadratico()
modelo.cargar_modelo(path_tf="modelo_entrenado.keras")

# Realizar predicciones
x_test = np.array([[0.5], [1.0], [1.5]])
predicciones = modelo.predecir(x_test)

print(predicciones)
# Output: [[0.25], [1.00], [2.25]] (aproximado a x²)


Opción 2: Entrenar un nuevo modelo
──────────────────────────────────
python
from modelo_cuadratico import ModeloCuadratico

modelo = ModeloCuadratico()
modelo.generar_datos(n_samples=1000, rango=(-1, 1), ruido=0.02)
modelo.construir_modelo()
modelo.entrenar(epochs=100, batch_size=32)
modelo.guardar_modelo("mi_modelo.keras", "mi_modelo.pkl")
